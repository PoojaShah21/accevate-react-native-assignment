# Accevate React Native Assignment

## Features
- Login with username & password
- OTP verification with 6-digit input
- Secure token storage
- Dashboard with dynamic background color
- Logout functionality

## Tech Stack
- React Native (TypeScript)
- React Navigation
- Axios
- AsyncStorage

## Setup Instructions
1. npm install
2. npx react-native run-android

## APK
Release APK is shared separately via email.

## Notes
- Accevate logo is downloaded from the official website and used as a local asset.
- Dashboard background color updates on every API hit as per requirement.
